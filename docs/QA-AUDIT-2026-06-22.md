# Lancer iOS — QA sweep & fixes (2026-06-22)

Simulator walk (iPhone 17 Pro, iOS 27) across onboarding + every primary surface, driving real taps
via XCUITest (the only thing that fires SwiftUI `Button` actions on this sim — `idb` can't) plus
screenshots. Verification baseline: full `LancerUITests` suite **10 passed / 2 skipped (live-relay
opt-ins) / 0 failed**; `build_sim` clean; `daemon/lancerd` `go test ./...` green.
