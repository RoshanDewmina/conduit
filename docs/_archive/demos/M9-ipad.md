# M9 — iPad + External Keyboard Parity

Status: in progress
Created: 2026-05-24

## Goal
On iPad and any device with a hardware keyboard, the app feels like a
serious shell client: split view, full keyboard parity, drag-and-drop.

## Demo Script
1. iPad split view: Workspaces on left, active Session on right.
2. Hardware keyboard: Ctrl-C interrupts a streaming command.
3. Cmd-T opens a new session in a sheet on the right pane.
4. Option-← / Option-→ produce word-jump in the terminal.
5. Drag a file from Files.app into the composer → uploads via SFTP,
   inserts remote path (requires M7 SFTP).
