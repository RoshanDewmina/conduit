# Away Mode docs — archived 2026-07-06

**Owner decision (2026-07-06):** The "Away Mode" product/pricing pivot is descoped. It was never
validated against real customers and is no longer being pursued.

These files were the primary Away Mode strategy/spec artifacts from the 2026-07-04 brainstorm.
They are kept for historical record only — do not treat them as live product direction.

**What remains live:** Feature and wireframe work that used "Away" in feature names (e.g. Away
Digest, Away Launch Composer) may still appear in design artifacts as legacy labels; the underlying
capabilities (home ledger, launch composer, proof suite) are unchanged — only the Away Mode *pivot*
and paid-workflow framing are retired. See `docs/product/2026-07-05-lancer-feature-master-plan.md`
for current scope (updated 2026-07-06 with descope note).

**Code check (2026-07-06):** No shipped Swift/Go implements Away Mode gating logic. The only code
references are mock UI strings (`"Away Mode Solo"`) in `CursorStyle/` demo shell files.
