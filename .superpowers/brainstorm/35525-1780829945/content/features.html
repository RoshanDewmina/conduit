Conduit — General-Purpose Agent Hub

What makes a dedicated app better than Discord/Telegram. Features mapped to what Hermes already does — just without a great mobile home.

💬 Agent Conversations Replaces Discord DMs
→ Thread-based chat per agent — context persists
→ Reply inline to any notification
→ Voice message to agent
→ Pre-built quick replies (shortcuts, not typing)
→ Natural language dispatch: "research TSLA and summarize"
→ Agent personas — Sakuya, researcher, concise… per agent
Now: You DM Hermes in Discord and hope it replies. No history, no context between sessions.
🗓️ Scheduling & Automation Replaces Hermes cron
→ Visual scheduler — "run trading agent at 9am ET weekdays"
→ Pause / resume / edit scheduled jobs from phone
→ Chain triggers — agent A finishes → agent B starts
→ Conditional triggers — "run when market opens"
→ Run history with outputs in-app
Now: Hermes cron jobs live in jobs.json. Paused Daily AI Briefing has been failing silently for weeks. No mobile visibility.
🏠 Daily Rhythm (native) Replaces Telegram delivery
→ Morning plan as an interactive card — tap to start each item
→ Evening review with simple mood + completion input
→ Daily briefing as rich in-app cards (not 6 bullets in Telegram)
→ Habit/streak widget on lock screen
→ "What should I focus on right now?" — natural language

Why not just use Discord / Telegram / WhatsApp

Plain text only
A trading agent says "Bought AAPL at $201.50" in a chat bubble. No chart, no rationale, no 1-tap undo.
No action buttons
To approve something you type "/approve 4821". In Conduit you tap. On Watch you crown-scroll and tap.
No notification control
Discord "mutes" or "unmutes" a channel. No rules, no priority tiers, no digest, no quiet hours per agent.
No agent awareness
Discord doesn't know what your agent is doing. Can't show fleet status, cost, health, or cross-agent context.

Suggested build order — what makes someone switch from Telegram immediately

1. Rich agent cards + inline actions
2. No-code notification rules
3. Conversation threads per agent
4. Visual scheduler (replace Hermes cron)
5. Daily rhythm + briefing as native cards
6. Agent marketplace
